# Installing

Currently **modleR** can be installed from GitHub:

``` r

# Without vignette

if(!require(devtools)){
    install.packages("devtools")
}

if(!require(gridpckgname)){
    devtools::install_github("Tai-Rocha/gridpckgname")
}

```
